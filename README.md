# DSA Visualizer

A terminal-based visualizer for sorting algorithms, trees, graphs, and pathfinding.

## Algorithms implemented
- **Sorting:** Bubble, Selection, Merge, Quick Sort
- **Trees:** Binary Search Tree (insert, search, traversal)
- **Graphs:** BFS, DFS
- **Pathfinding:** Dijkstra's algorithm

## Setup
```bash
pip install -r requirements.txt
python main.py
```

## Time complexity
| Algorithm     | Best     | Average  | Worst    |
|---------------|----------|----------|----------|
| Bubble Sort   | O(n)     | O(n²)    | O(n²)    |
| Merge Sort    | O(n log n) | O(n log n) | O(n log n) |
| Dijkstra      | O(E log V) | O(E log V) | O(E log V) |
